# 项目简介

本项目是基于深度学习的非侵入式负荷监测系统，旨在通过智能算法实现家庭或工业用电负荷的实时监测与分析。系统主要包含数据采集、特征提取、负荷识别及能耗分析等模块。技术特点在于运用了卷积神经网络和循环神经网络，无需物理接触即可准确识别电器状态与能耗。此系统有助于提高能效管理，促进节能减排，具有良好的应用前景。


## 核心功能
- 实时监测家庭或工业用电负荷
- 数据采集模块
- 特征提取模块
- 负荷识别模块
- 能耗分析模块

## 适用场景
- 家庭用电监测与分析
- 工业用电监测与分析
- 能效管理
- 节能减排

## 技术特点
- 基于深度学习算法
- 运用卷积神经网络(CNN)
- 运用循环神经网络(RNN)
- 非侵入式监测技术，无需物理接触
- 准确识别电器状态与能耗
